### ВВЕДЕНИЕ
В современном мире цифровизация образования становится не просто трендом, а необходимостью. Стереометрия — раздел геометрии, изучающий фигуры в пространстве, традиционно считается одним из самых сложных для восприятия школьниками и студентами из-за необходимости развитого пространственного мышления. 

**Актуальность работы** заключается в создании доступного и интерактивного инструмента, который позволяет визуализировать сложные геометрические концепции и мотивировать учащихся через систему достижений и баллов.

**Цель работы** — разработка полноценного веб-ресурса, объединяющего теоретический видеоматериал, практические задания и систему отслеживания прогресса.

### ГЛАВА 1. ТЕОРЕТИЧЕСКАЯ ЧАСТЬ
#### 1.1. Роль электронных образовательных ресурсов в обучении геометрии
Электронные образовательные ресурсы (ЭОР) позволяют перейти от пассивного поглощения информации к активному взаимодействию. В изучении стереометрии ЭОР решают проблему наглядности, предоставляя возможность просмотра видеоматериалов и интерактивных моделей.

#### 1.2. Обзор существующих платформ
Анализ таких систем, как Moodle, Google Classroom и Khan Academy, показывает, что несмотря на их мощный функционал, зачастую не хватает узкоспециализированных инструментов для конкретных предметов (например, для проверки задач по стереометрии) и вовлекающих механик. Использование Node.js позволяет создать легковесное и быстрое решение, адаптированное под нужды конкретного учебного курса.

---

### ГЛАВА 2. РАЗРАБОТКА ВЕБ-ПЛАТФОРМЫ «STEREOMETRYONLINE»
#### 2.1. Архитектура и стек технологий проекта
Для реализации проекта был выбран современный стек:
- **Backend:** Node.js с фреймворком Express.js для обработки API-запросов.
- **База данных:** SQLite (для локальной разработки) и PostgreSQL (для продакшн-среды) с использованием ORM-подобного подхода.
- **Frontend:** Чистый HTML, CSS и JavaScript для обеспечения максимальной производительности и кроссбраузерности.
- **Безопасность:** JWT (JSON Web Tokens) для управления сессиями и BcryptJS для защиты паролей пользователей.

#### 2.2. Реализация функциональных модулей
Проект разделен на несколько ключевых роутов:
- `/api/auth`: Регистрация и вход с хешированием паролей.
- `/api/videos`: Управление видеоконтентом, разделенным на секции (например, «Основы», «Призмы»).
- `/api/assignments`: Система теоретических и практических заданий.
- `/api/admin`: Панель управления для преподавателя, позволяющая добавлять контент и проверять работы.

#### 2.3. Система геймификации и личный кабинет
Особенностью платформы является модуль `gamification.js`. Пользователи получают баллы за:
- Просмотр обучающих видео.
- Успешную сдачу заданий.
- Активность в комментариях.
При достижении определенных порогов пользователю открываются «достижения» (achievements), что повышает уровень вовлеченности (engagement).
