СТРУКТУРНОЕ ОПИСАНИЕ

Программы для ЭВМ: «Электронный образовательный ресурс по стереометрии с элементами геймификации «StereometryOnline»»

1. ОБЩИЕ СВЕДЕНИЯ
Электронный образовательный ресурс (ЭОР) «StereometryOnline» представляет собой современную веб-платформу, предназначенную для обучения учащихся основам стереометрии. Система объединяет в себе теоретический материал в формате видеоуроков, систему практических заданий и элементы игрового обучения для повышения мотивации учащихся.

2. АРХИТЕКТУРА СИСТЕМЫ
Программа построена по архитектуре клиент-серверного веб-приложения:
- Backend: Среда выполнения Node.js с использованием фреймворка Express.js. Обеспечивает работу REST API, управление базой данных и аутентификацию.
- Frontend: Веб-интерфейс на базе HTML5, CSS3 и JavaScript (ES6+), обеспечивающий интерактивное взаимодействие с пользователем.
- База данных: Реляционная СУБД (SQLite/PostgreSQL) для хранения данных о пользователях, контенте, прогрессе и достижениях.

3.1. Главная страница (Landing Page)
Содержит общую информацию о платформе, кнопка начала обучения и навигационное меню.

4. АЛГОРИТМ РАБОТЫ ПРОГРАММЫ
1. Запуск сервера: `npm start` инициализирует базу данных и запускает HTTP-сервер на порту 3000.
2. Авторизация: Пользователь вводит учетные данные, сервер проверяет их и выдает токен доступа.
3. Обучение: Пользователь выбирает раздел, смотрит видео. При завершении видео в БД фиксируется прогресс и начисляются баллы.
4. Выполнение заданий: Учащийся отправляет решение задания. Преподаватель через админ-панель проверяет работу.
5. Геймификация: Система автоматически проверяет условия выдачи достижений (например, «Посмотрел 5 видео») и обновляет профиль пользователя.

5. ТЕХНИЧЕСКИЕ ТРЕБОВАНИЯ
- Сервер: Node.js v14+, 512 МБ ОЗУ.
- Клиент: Любой современный веб-браузер (Chrome, Firefox, Safari).
- СУБД: SQLite или PostgreSQL.
